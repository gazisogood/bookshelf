# bookshelf
Веб-сервис для аренды и бронирования книг.  

### Задача
Разработка веб-сервеса для аренды книг из школьной библиотеки.

Основные возможности:
- Поиск книги по каталогу
- Аренда книги
- Добавление новой книги в библиотеку 

### Stack
- React
- Firebase
